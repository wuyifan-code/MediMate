# 打包APK计划

## 1. 构建React应用
- 执行 `npm run build` 命令，生成dist目录
- 这将把React应用编译成静态文件，用于后续打包

## 2. 同步Web应用到Android项目
- 执行 `npx cap sync android` 命令
- 这将把dist目录中的静态文件复制到Android项目的assets目录
- 同时更新Android项目的配置和依赖

## 3. 构建Android APK
- 进入android目录：`cd android`
- 执行 `./gradlew assembleRelease` 命令（Windows上使用 `gradlew.bat assembleRelease`）
- 这将生成Release版本的APK文件
- APK文件将位于 `android/app/build/outputs/apk/release/` 目录

## 4. 验证APK文件
- 检查生成的APK文件是否存在
- 可选：使用 `adb install` 命令安装到设备进行测试

## 环境要求检查结果
- ✅ Node.js v24.11.1 已安装
- ✅ npm 11.6.2 已安装
- ✅ Java JDK 17.0.17 已安装
- ✅ ANDROID_HOME 已配置
- ✅ JAVA_HOME 已配置
- ✅ Android SDK 已安装
- ✅ 项目依赖已安装

所有环境要求都已满足，可以直接执行打包流程。
